# Contributing to Sunny Acres Farm Records

Thank you for helping maintain our farm records! This guide explains
how to contribute updates and improvements.

## How to Contribute

### 1. Create an Issue First

Before making changes, create an Issue describing what you want to
update. This helps the team know what's being worked on.

### 2. Fork the Repository

Click the **Fork** button to create your own copy of this repository.

### 3. Make Your Changes

Edit files directly on GitHub:

1. Navigate to the file you want to change
2. Click the pencil icon (Edit)
3. Make your updates
4. Write a clear commit message explaining what changed

### 4. Submit a Pull Request

1. Go to the **Pull requests** tab
2. Click **New pull request**
3. Select your fork and branch
4. Fill out the PR template
5. Submit for review

## What to Contribute

- Updated planting schedules and crop records
- Equipment maintenance logs
- Livestock health observations
- Compliance documentation updates
- Corrections to existing information

## Guidelines

- **Be accurate:** Double-check dates, numbers, and details
- **Be specific:** Include field names, animal IDs, equipment models
- **Be timely:** Update records as soon as practical after events
- **Add photos:** When helpful, attach images to Issues

## Questions?

Open an Issue with the "question" label or start a Discussion.
