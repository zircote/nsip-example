# Getting Help

## Where to Get Support

### For Farm Operations Questions

- Create an Issue with the appropriate label
- Start a Discussion for general questions
- @mention a team member who can help

### For GitHub Questions

- Check the session reference guides in the training materials
- Ask your instructor or training coordinator
- Review GitHub's official documentation at docs.github.com

## Quick Links

| Need | Action |
|------|--------|
| Report equipment issue | Create Issue with `equipment` label |
| Log crop observation | Create Issue with `crops` label |
| Record livestock health | Create Issue with `livestock` label |
| Ask a general question | Start a Discussion |
| Suggest an improvement | Create Issue with `planning` label |

## Response Times

- **Urgent issues:** Same day during business hours
- **Routine issues:** Within 2-3 business days
- **Discussions:** As time permits
