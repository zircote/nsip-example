# Security Policy

## Overview

This repository contains farm management records and documentation.
While not software code, we take data security seriously.

## Sensitive Information

**Do NOT include in this repository:**

- Social Security numbers or tax IDs
- Bank account or financial account numbers
- Passwords or access credentials
- Medical records (human)
- Detailed personal contact information of minors

**Acceptable to include:**

- General farm location (county level)
- Crop and livestock records
- Equipment information
- Business contact information
- Compliance documentation (redacted as needed)

## Reporting Security Concerns

If you notice sensitive information that should not be public:

1. **Do not create a public Issue**
2. Contact the repository owner directly
3. Describe what information needs to be removed
4. We will address it promptly

## Access Control

- Repository collaborators should only be farm team members
- Review collaborator list periodically
- Remove access promptly when team members leave

## Best Practices

- Review files before committing to ensure no sensitive data
- Use general descriptions rather than specific personal details
- When in doubt, ask before posting
